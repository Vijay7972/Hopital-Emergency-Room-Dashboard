# 🏥 Hospital Emergency Room Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Hospital Emergency Room Dashboard** built in **Power BI** to analyze patient visits, waiting times, satisfaction scores, admission rates, department referrals, and overall ER performance.

The dashboard helps hospital administrators and healthcare professionals monitor operational efficiency, identify bottlenecks, and make data-driven decisions to improve patient care.

---

## 🎯 Project Objectives

- Monitor Emergency Room performance
- Track patient admissions and referrals
- Analyze patient waiting time
- Measure patient satisfaction
- Identify peak patient visits
- Improve resource planning using data insights

---

## 📊 Dashboard Pages

### 1. Monthly View
Provides a month-wise overview of:

- Total Patients
- Average Wait Time
- Patient Satisfaction Score
- Admission Rate
- Gender Distribution
- Age Group Analysis
- Race Distribution
- Department Referrals
- Hourly Patient Trend
- Daily Patient Trend

---

### 2. Consolidated View

Displays overall hospital performance including:

- Overall KPIs
- Patient Trends
- Wait Time Analysis
- Department Referral Analysis
- Admission Analysis
- Demographic Insights

---

### 3. Patient Details

Detailed patient-level information including:

- Patient ID
- Admission Date
- Gender
- Age
- Race
- Department Referral
- Admission Status
- Satisfaction Score
- Wait Time

Useful for operational monitoring and detailed investigation.

---

### 4. Key Takeaways

A summary page highlighting:

- Important KPIs
- Performance Insights
- Operational Trends
- Areas requiring improvement
